# Sticky Notes

A simple Django sticky notes web application built for a HyperionDev bootcamp task.

## Features
- Create notes
- View notes
- Edit notes
- Delete notes

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS

## How to Run
1. Clone the repository
2. Navigate to project folder
3. Run migrations:
   python manage.py makemigrations
   python manage.py migrate
4. Start server:
   python manage.py runserver
